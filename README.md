# 포켓몬 도감 (Poke Dex)

---

## 홈 (Home) 페이지

사용자에게 포켓몬 검색 및 주요 정보를 빠르게 제공하는 페이지 입니다.

### 1. 포켓몬 검색 영역

메인 기능인 포켓몬 검색을 위한 핵심 영역입니다.

검색을 실행하면 검색(Search) 페이지로 이동하여 포켓몬을 검색합니다.

### 2. 오늘의 포켓몬 영역

사용자의 흥미를 유발하는 일일 추천 포켓몬 정보 영역입니다.

추천 포켓몬의 도감번호, 이름, 타입을 표시하며,
자세히 보기 버튼으로 해당포켓몬을 즉시 검색할 수 있습니다.

### 3. 최근 검색 영역

사용자가 최근에 검색했던 포켓몬 기록을 빠르게 재탐색할 수 있는 영역입니다.

최대 10개의 최근 검색을 기록하며, 클릭하여 즉시 검색할 수 있습니다.

---

## 검색 (Search) 페이지

사용자가 검색한 특정 포켓몬의 모든 상세 정보를 집중적으로 표시하는 페이지 입니다.

### 1. 검색 바

페이지 상단에서 새로운 포켓몬을 검색할 수 있는 영역입니다.

### 2. 기본 정보

검색한 포켓몬의 기본 스프라이트와 색이 다른 스프라이트, 이름, 도감번호, 타입을 표시합니다.

### 3. 진화 계열

검색한 포켓몬의 진화체와 특수폼을 모두 표시합니다.

클릭하여 특정 포켓몬을 재검색 할 수 있습니다.

### 4. 종족값 및 특성

검색한 포켓몬의 종족값, 특성, 포획률을 표시합니다.

### 5. 타입 상성

검색한 포켓몬의 타입 상성을 표시합니다.

특정 타입을 클릭하여 타입(Type) 페이지로 이동할 수 있습니다.

---

## 타입 (Type) 페이지

사용자가 선택한 특정 타입이 공격할 때, 방어할 때 상성을 표시하는 페이지 입니다.

타입 리스트에서 타입을 선택하면 상성이 표시됩니다.

---

## 메모 (Memo) 페이지

사용자가 간단한 메모를 할 수 있는 페이지 입니다.
